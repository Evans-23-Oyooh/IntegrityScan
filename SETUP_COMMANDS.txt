# IntegrityScan - Copy & Paste Setup Commands

## For Windows Command Prompt

### Option 1: Automated Setup (Recommended)
```
setup_complete.bat
```

### Option 2: Manual Setup

**Step 1: Install Dependencies**
```
pip install -r requirements.txt
```

**Step 2: Download NLTK Data**
```
python -c "import nltk; nltk.download('punkt'); nltk.download('stopwords'); nltk.download('wordnet'); nltk.download('punkt_tab')"
```

**Step 3: Run Migrations**
```
python manage.py migrate
```

**Step 4: Collect Static Files**
```
python manage.py collectstatic --noinput
```

**Step 5: Run Development Server**
```
python manage.py runserver
```

**Step 6: Open Browser**
```
http://localhost:8000
```

## Troubleshooting Commands

### If transformers fails to install:
```
pip install transformers==4.30.0 --no-cache-dir
pip install torch==2.0.0 --no-cache-dir
```

### If migrations fail:
```
python manage.py migrate --run-syncdb
```

### If static files fail:
```
python manage.py collectstatic --noinput --clear
```

### If port 8000 is in use:
```
python manage.py runserver 8001
```

### To create superuser:
```
python manage.py createsuperuser
```

### To check Django version:
```
python -c "import django; print(django.VERSION)"
```

### To verify all dependencies:
```
python -c "import django, nltk, transformers, torch; print('All OK')"
```
